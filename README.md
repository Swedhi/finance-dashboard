# Finflow — Finance Dashboard

A responsive finance dashboard built with React and CSS.

## How to Run
Open `index.html` in any browser. No installation needed.

## Live Demo
https://yourusername.github.io/finflow-dashboard

## Features
- Dashboard with balance trend and spending charts
- Transactions table with search, filter, and sort
- Add / Delete transactions (Admin role)
- Insights with spending breakdown and monthly comparison
- Role-based UI — Admin and Viewer modes
- Data persists via localStorage
- Fully responsive design

## Tech Used
- React 18 (via CDN)
- Chart.js 4
- Pure CSS (dark theme)
